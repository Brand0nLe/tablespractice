# Tables (Angular Material Table) Practice w/ Custom Paginator

This is just for practice attempting to create a Table using Angular Material's table, but with a custom paginator.

## Setup

To run this project, follow these steps:

1. Clone the repository to your local machine
2. Install dependencies by running `npm install` in the project directory
3. Start the development server by running `npm start`
4. Open `http://localhost:3000` in your web browser to view the site
